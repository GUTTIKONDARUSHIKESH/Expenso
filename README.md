# Expenso – Live Expense Tracker & Analytics Dashboard

## Overview
Expenso is a real-time expense tracking and analytics dashboard built using Python and Streamlit.  
It allows users to record daily expenses and analyze spending patterns through charts.

## Features
- Add daily expenses
- Live expense tracking
- Expense table view
- Total expense metric
- Category-wise bar chart
- Expense distribution pie chart
- Daily spending trend line graph
- Download expense report (CSV)

## Technologies Used
- Python
- Pandas
- Streamlit
- Matplotlib

## How to Run the Project

1. Install required libraries:
```
pip install streamlit pandas matplotlib
```

2. Run the dashboard:
```
python -m streamlit run dashboard.py
```

3. Open in browser:
```
http://localhost:8501
```
